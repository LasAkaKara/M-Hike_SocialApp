## Responsive Design

### Supports Multiple Screen Sizes
- **Small phones** (4.5"): Single column lists
- **Regular phones** (5-6"): Single column, optimized padding
- **Tablets** (7+") : Multi-column layout (prepared for Task F)

All layouts use:
- `android:layout_weight` for flexible spacing
- `ConstraintLayout` for responsive positioning
- `match_parent` and `wrap_content` appropriately
- Padding/margins that scale with density
